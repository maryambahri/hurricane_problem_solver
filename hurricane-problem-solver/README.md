# Hurricane Problem Solver

A reusable skill for deep root-cause analysis in messy, high-stakes, deceptively complex situations.

An open-source skill by [AI Arm](https://www.aiarm.sa).

## What it does

This skill helps an agent:

- turn a raw complaint into a sharper problem statement
- map symptoms before forcing a story
- open multiple hypotheses in parallel
- follow hidden constraints, incentives, patterns, and chain reactions
- separate symptom, driver, and root mechanism
- identify the highest-leverage intervention

The style is intentionally nonlinear, investigative, and unsentimental. It is built for cases where the visible issue is probably not the real issue.

## Best use cases

Use this skill when:

- a problem feels bigger than it should
- symptoms are multiplying
- the obvious explanation feels too neat
- several issues may actually share one upstream cause
- you need root-cause analysis instead of surface troubleshooting
