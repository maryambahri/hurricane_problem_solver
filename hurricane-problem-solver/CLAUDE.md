# CLAUDE.md

This repository contains a reusable Claude skill.

## Skill location

Use this folder:

```text
skills/hurricane-problem-solver/
```

## Files

```text
skills/
  hurricane-problem-solver/
    SKILL.md
    references/
      questioning-patterns.md
```

## Purpose

`hurricane-problem-solver` is a deep root-cause analysis skill for messy, high-stakes, deceptively complex problems.

Use it when:
- the visible issue is probably not the real issue
- symptoms are multiplying
- the first neat explanation feels incomplete
- you need structural diagnosis, not surface troubleshooting

## Installation

Copy `skills/hurricane-problem-solver/` into your Claude skills directory.

## Usage note

Treat `SKILL.md` as the primary instruction file.
Read `references/questioning-patterns.md` only when the problem is still fuzzy or needs harder pressure-testing from unusual angles.
